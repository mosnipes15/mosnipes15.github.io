My Projects

A selection of my recent work, spanning UX analysis, creative design, hardware assembly, database implementation, and basic game development.

Coffee Shop Database Schema

This is a coffee shop database schema that was created in collaboration with other peers as part of a final project in ITSC 3160 (Database Design & Implementation). The goal was to demonstrate mastery of database implementation and SQL querying for business management.

Lights Off! Game Project

This is an interactive puzzle game that was developed in Java during my time in ITSC 2214 (Data Structures & Algorithms). It was designed to showcase mastery in boolean logic and core game design mechanics. The primary objective of the player is to extinguish all active lights on the board, but it comes with a challenge! It features a toggling mechanic, whereas interacting with any single light simultaneously inverts the on/off state of its four adjacent lights.

My Gaming PC Build

This is an image of my gaming desktop computer that I built during New Year's Eve. It features many high-end components, including an AIO liquid CPU cooler that features a digital temperature display screen, a Zotac Gaming GeForce RTX 5080 graphics card, and 32 GB of illuminated Corsair memory modules. It is my favorite project to date for many reasons, but mostly because I love demonstrating my prowess with hardware assembly.

SQL Query Design

This is a screenshot of a query design assignment created using mySQL Workbench that I completed in ITSC 3160 (Database Design & Implementation). The goal of this project was to demonstrate mastery of modifying relational schemas, enforcing strict data integrity, and building tables that are self-referencing. This assignment highlights the approach that I took to designing robust, well-structured databases that prioritizes clean data management.
